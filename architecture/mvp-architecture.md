## 1. Клиентский слой

| Что | Как |
|---|---|
| Web-приложение | React + Vite |
| Карта | Yandex Maps JS API |
| Список магазинов | Фильтры: по близости, по релевантности MCC |
| Голосование | Один клик — выбор MCC-кода |
| Категории кэшбэка | Пользователь выбирает свои банки/категории для персонализации |
| Мобильное приложение | Android — Phase 2 (тот же REST API) |

---

## 2. Интеграционный слой

**Yandex Maps JS API** — основной и единственный источник каталога магазинов.

- Весь каталог торговых точек **не хранится** локально
- У себя в БД храним только **обогащённые данные**: проголосованные MCC, статистику, связи
- Магазины идентифицируются по `yandex_firm_id`

---

## 3. Backend слой

**Oracle Compute VM (Always Free) — Node.js / Express**

| Сервис | Ответственность |
|---|---|
| Сервис магазинов | Поиск по геопозиции, фильтрация, связь с Yandex firm_id |
| Сервис голосования | Запись голоса, дедупликация, обновление агрегатов |
| Сервис локаций | Расчёт расстояний, геофильтры |
| Сервис пользователей | Регистрация, авторизация, профиль |
| Сервис статистики | Расчёт top_mcc, last_mcc, популярность по периодам |

**Планируется:** Redis для кэширования топов и геозапросов.

---

## 4. Data слой

**Oracle Autonomous Database (Always Free, 20 GB)**

Хранит только обогащённые данные — то, чего нет у Яндекса:

| Таблица | Содержимое |
|---|---|
| `merchants` | Связь `yandex_firm_id` → наши данные (подтверждённый MCC и т.д.) |
| `mcc_votes` | История всех голосов пользователей |
| `users` | Аккаунты, настройки кэшбэка |
| Агрегаты | `top_mcc`, `last_mcc`, счётчики по периодам |

---

## 5. Монетизация (Phase 2+)

- Реклама на карте и в списке магазинов
- Платное продвижение сетевых магазинов
- Возможная интеграция каталогов сетей напрямую

---

## Инфраструктура Oracle Cloud Free Tier

| Ресурс | Использование |
|---|---|
| Compute VM (AMD E2.1.Micro × 2) | Node.js + React (оба на одной VM на старте) |
| Autonomous Database | Основное хранилище данных |
| Object Storage (20 GB) | Статика, резервные копии |
| Outbound трафик (10 TB/мес) | API-ответы, карта |
