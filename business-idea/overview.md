# MCC Tracker — Краудсорсинг MCC-кодов для максимизации кэшбека

## Описание
Сервис для отслеживания актуальных MCC-кодов (Merchant Category Code) торговых точек, чтобы пользователи могли правильно планировать покупки и выбирать нужные категории кэшбека в банках.

## Проблема
В России банки начисляют кэшбек по категориям, определяемым MCC-кодом магазина. Проблема в том, что:
- один и тот же магазин может проходить по разным MCC-кодам в разные дни/недели/месяцы
- пользователь заранее не знает, какой код присвоен конкретной торговой точке сегодня
- выбранная категория кэшбека может не совпасть с фактическим кодом — и кэшбек не начислится

## Решение
Краудсорсинговая платформа, где пользователи после покупки оставляют MCC-код, по которому прошёл платёж. Система агрегирует данные и показывает:
- **последний актуальный код** по магазину
- **самый популярный код за день** (по голосам пользователей)

## Ценностное предложение
- Пользователи получают актуальные данные о MCC-кодах и не теряют кэшбек
- Чем больше пользователей — тем точнее данные (сетевой эффект)
- Первая база данных магазинов берётся с карты Яндекса

## Целевая аудитория
- Активные пользователи карт с кэшбеком (Тинькофф, Альфа, Сбер и др.)
- Люди, которые осознанно планируют расходы и хотят получать максимум от банковских программ лояльности

## Ключевые функции (MVP)
- [ ] Список и карта торговых точек
- [ ] Фильтрация по релевантности и по близости (геолокация)
- [ ] Авторизация пользователя
- [ ] Простановка MCC-кода кликом по магазину
- [ ] Отображение последнего и самого популярного кода за день

## Дорожная карта
1. **Web-версия** — список + карта, краудсорсинг кодов
2. **Android-приложение** — мобильный доступ, геолокация
3. **Монетизация** — реклама + каталоги сетевых магазинов
